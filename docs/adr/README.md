# Architecture Decision Records (ADR)

Сюда фиксируются исключительно ключевые (тектонические) технические решения, которые ИИ-агент **не должен пересматривать без явного указания владельца**. Формат краткий: **Контекст → Решение → Последствия**. Мелкие решения в ADR не пишем (анти-doc-churn, см. `../../AGENTS.md` §10).

## Активные ADR
- `0001-whatsapp-channel.md` — выбор WhatsApp-канала и модель владения процессами.
- `0002-payment-gateway.md` — выбор платёжного провайдера (Merchant of Record).
